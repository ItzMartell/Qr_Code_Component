# 🧩 Proyecto: Componente QR Code

Este proyecto consiste en el desarrollo de un **componente de Código QR** utilizando **Astro** y **Tailwind CSS**.  
El objetivo es aplicar los conocimientos sobre **componentes**, **maquetación**, **estilos responsivos** y **utilidades CSS** para construir un diseño limpio, moderno y adaptable a diferentes dispositivos.

---

## 📖 Descripción general
En esta actividad se desarrollará un componente de Código QR utilizando Astro y Tailwind CSS, con el objetivo de poner en práctica los conocimientos en la creación de componentes, uso de estilos y diseño responsivo.

### 💡 Lo que aprendí
En este proyecto aprendí a estructurar una página web de manera más organizada, separando los componentes para hacer el código más limpio y reutilizable. También comprendí cómo funciona el manejo de archivos estáticos, especialmente el uso de la carpeta public para diferenciar entre usar rutas directas e importar recursos desde otras carpetas.

En general, este proyecto me ayudó a familiarizarme con Astro, mejorar mi lógica al resolver errores y fortalecer mis habilidades en el desarrollo frontend.
